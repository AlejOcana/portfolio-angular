# portfolio-angular

Personal portfolio built with Angular 21, TypeScript, and SCSS.

## Tech Stack
- **Framework:** Angular 21
- **Language:** TypeScript
- **Styling:** SCSS with CSS variables
- **Testing:** Vitest (unit) + Playwright (E2E)

## Commands
```bash
npm start          # Dev server: localhost:4200
npm run build     # Production build
npm run test:unit # Unit tests (Vitest)
npm run test:e2e  # E2E tests (Playwright)
npm run test:e2e:ui # E2E tests with UI
```

## Features
- Fully responsive (mobile-first)
- Dark/light mode (auto-detects system preference)
- Accessible (keyboard navigation, ARIA labels, focus states)
- Mobile hamburger menu with backdrop
- Scroll-to-top button

## Sections
- Hero with terminal-style code display
- Impact metrics
- About with current role and focus areas
- Technical approach
- Work experience
- Education & certifications
- Featured projects
- Tech stack (Frontend, Backend, Databases, DevOps, Learning)
- Footer with social links
